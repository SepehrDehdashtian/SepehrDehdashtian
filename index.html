Framework (NIFTY)

  • Perturbing Node Attributes
\[ \begin{aligned} \mathbf{\tilde{x}}_u = \mathbf{x}_u + \mathbf{r} \circ \delta \end{aligned} \] $\delta$: M-dim vector sampled from normal distribution
$\mathbf{r}$: M-dim vector sampled from Bernouli distribution

Framework (NIFTY)

  • Perturbing Sensitive Attribute
\[ \begin{aligned} \mathbf{u} \longrightarrow \mathbf{\tilde{u}}_s \end{aligned} \]

Framework (NIFTY)

  • Perturbing Graph Structure
\[ \begin{aligned} \mathbf{\tilde{A}} = \mathbf{A} \circ \mathbf{R}_e \end{aligned} \] $\mathbf{A}$: Adjacency Matrix
$\mathbf{R}_e$: $N \times N$ matrix sampled from Bernouli distribution

Framework (NIFTY)

  • Fairness and Stability Objective Function
\[ \begin{aligned} \mathcal{L}_s = \mathbb{E}_u \left[ \frac{1}{2} \left( D\left( t \left( \mathbf{z}_u \right) , sg \left( \mathbf{\tilde{z}}_u \right) \right) + D\left( t \left( \mathbf{\tilde{z}}_u \right) , sg \left( \mathbf{z}_u \right) \right) \right) \right] \end{aligned} \] $D$: Cosine Distance
$t$: $\mathbb{R}^d \longrightarrow \mathbb{R}^d$ transformer (MLP)
$\mathbf{z}_u$: Representation of node $u$
$\mathbf{\tilde{z}}_u$: Representation of perturbed node $u$
$sg$: Stopgrad

Framework (NIFTY)

  • Overall Objective Function
\[ \begin{aligned} \underset{\theta_{Enc}, \theta_t, \theta_f}{\text{min}} \mathbb{E}_u \left[ (1 - \lambda) \mathcal{L}_c \right] + \lambda \mathcal{L}_s \end{aligned} \] $\mathcal{L}_c$: Node Classification Loss
$\mathcal{L}_s$: Fairness and Stability Loss
$\lambda$: Regularization Coefficient
